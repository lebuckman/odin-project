# Admin Dashboard

This project puts newly-learned CSS Grid skills to the test.

Note that while the `Your Projects` grid is responsive, the page as a whole is not yet responsive to different screen sizes.

Learn more about this project in [TOP's Intermediate HTML and CSS Course: Grid](https://www.theodinproject.com/lessons/node-path-intermediate-html-and-css-admin-dashboard).
