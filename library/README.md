# The Odin Project: Library Project

A simple Library app that supplements learning about JavaScript objects, object constructors, and the `[[Prototype]]`. Learn more about the project in [TOP's JavaScript Course](https://www.theodinproject.com/lessons/node-path-javascript-library).

**Additional Misc. Topics Reviewed:**

-   Manipulating the DOM with JS
-   Event delegation
-   Working with forms inside the HTML `dialog` tag

## Usage

-   Starts with 5 books displayed.
