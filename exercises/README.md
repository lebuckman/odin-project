# Exercises

This directory contains all exercises from various points in The Odin Project curriculum.

Initially, these exercises were forked repos, but merging these forked repos into this collection repo meant that the entire git history of the forked repos were also pulled (`git subtree`), hence the large quantity of commits and contributors.

Linked below are the original, up-to-date Odin repositories:

- [`css-exercises`](https://github.com/TheOdinProject/css-exercises)
- [`javascript-exercises`](https://github.com/TheOdinProject/javascript-exercises)
